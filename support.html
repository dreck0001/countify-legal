Countify Support

Frequently Asked Questions

What is Countify?

Countify is a simple yet powerful counter app for iOS that allows you to create and manage multiple counters for different purposes. Whether you're counting steps, tracking inventory, or keeping score in a game, Countify makes it easy with a clean, intuitive interface.

How do I create a new counter?

Tap the "+" button at the bottom of the main screen to create a new counter. You can customize the name, step size, limits, and haptic feedback settings to suit your needs.

What does "Step Size" mean?

Step size determines how much the counter value changes with each increment or decrement. For example, if you set a step size of 5, each tap will increase or decrease the count by 5 instead of 1.

Can I set limits on my counters?

Yes! You can set upper and lower limits for any counter. Once a limit is reached, the app will prevent you from going beyond that value. This is useful for tracking items with a fixed quantity or for creating percentage-based counters.

What is haptic feedback?

Haptic feedback provides subtle vibrations when you interact with the app. In Countify, you'll feel different vibration patterns when incrementing, decrementing, or resetting a counter. This feature can be enabled or disabled for each counter.

How do I reset a counter?

To reset a counter, open the counter you want to reset, tap the menu button (three dots) in the top right corner, and select "Reset Counter."

How do I delete a counter?

There are two ways to delete a counter:

  1. From the main screen, long-press on a counter and select "Delete" from the menu.
  2. While viewing a counter, tap the menu button (three dots) in the top right corner and select "Delete Counter."
